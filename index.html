What you'll find here

This is a simple, accessible homepage created for the My_First_Portfolio project. It demonstrates headings, paragraphs, navigation and an accessible clickable image. Use it as the starting point or replace sections to match your own content.

The page includes semantic structure — header, nav and main — so screen readers and search engines can understand the layout. Keep your content clear and use <h1>, <h2> and <h3> for headings in proper order.

Below is a clickable image. The image itself has descriptive alternative text for accessibility, and the link wraps the image so clicking the image will navigate to the linked site.

Feel free to edit this file inside the html_basic directory and push changes to your GitHub repository (My_First_Portfolio) when you're ready.
